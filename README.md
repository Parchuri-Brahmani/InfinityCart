# InfinityCart - E-Commerce Platform

InfinityCart is a comprehensive full-stack e-commerce solution designed for a seamless online shopping experience. This project integrates modern web technologies to handle product management, secure user authentication, and real-time user engagement features.

## 🌟 Key Features
- **Dynamic Product Catalog**: Organized categories for easy navigation and product discovery.
- **Real-Time Live Browser Count**: Tracks and displays active user sessions dynamically.
- **Smart Notification System**: Instant AJAX-based feedback for cart and wishlist actions.
- **Secure Authentication**: Robust login and registration system with encrypted password storage.
- **Responsive UI**: Fully optimized for various screen sizes (Mobile, Tablet, and Desktop).

## 🛠️ Technical Stack
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap
- **Backend**: PHP, Node.js (Express framework)
- **Database**: MySQL (Relational Database Management)
- **Deployment**: InfinityFree (Live Hosting)

## 📂 Project Structure
- `/admin`: Management dashboard for inventory and orders.
- `/css`: Custom stylesheets for the platform's aesthetics.
- `/images`: Media assets and product catalogs.
- `/php`: Server-side logic and database connection scripts.
